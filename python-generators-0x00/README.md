# alx-backend-python# Python Generators Assignment

This folder contains Python scripts for the ALX backend Python generators project:
- `seed.py` – creates database and tables, inserts sample data
- `0-stream_users.py` – generator to stream user rows one by one
- `1-batch_processing.py` – batch processing of users over age 25
- `2-lazy_paginate.py` – lazily loads paginated data
- `4-stream_ages.py` – memory-efficient aggregation of user ages
